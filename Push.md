# Belajar GitHub Dasar

Repository ini dibuat untuk memenuhi tugas pembelajaran penggunaan GitHub.

## Deskripsi
Project ini berisi contoh penggunaan GitHub seperti:
- Membuat repository
- Upload file
- Edit file langsung di GitHub
- Upload kode sederhana
- Membagikan link repository

## Isi Repository
- README.md → Penjelasan project
- contoh.txt → Contoh file teks
- index.html → Contoh kode sederhana

## Cara Menggunakan
1. Buka repository ini di GitHub
2. Pilih file yang ingin dilihat
3. Klik nama file untuk melihat isinya

## Tujuan
Melatih siswa agar memahami dasar penggunaan GitHub sebagai media penyimpanan dan kolaborasi kode secara online.
